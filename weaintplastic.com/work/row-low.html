Frontend Framework

Row Low

What has started with a really rough version back in the year 2013, finally grew up and became a dead simple fronted framework with mobile in mind. Row Low is not a UI framework, it simply gives you a handful of components to setup and structure your code to build responsive websites fast and clean.

Visit Project Website

I began working on Row Low back in 2013 since I've missed the functionality of available frameworks to nest responsive grids to an infinite level. What has started as a simple responsive grid framework evolved quickly to something like a personal toolkit for building responsive websites faster. Common and heavily used components were added to the framework one by one and were put to the test while using them in production. Early 2015 I decided to completely rewrite the framework, switching from LESS to SCSS, making it truly modular by shipping each component as a bower package and release it as an open-source project on github.
